## 確立したアーキテクチャ（Day64）

### 「鏡」設計
```
D&D Window スライダー操作 → engine.handleMidiCC() → plugin.params（SSoT）
MacroKnob ノブ操作       → engine.receiveMidiModulation() → plugin.params（SSoT）

plugin.params（SSoT）
  ↑↓ 200ms ポーリング（getParametersLive）
D&D Window スライダー（表示同期）
```

- MacroKnob と D&D Window は**直接交信しない**
- `plugin.params` だけを見ている「鏡」
- `engine.ts` は無変更
