## Day64 で完了したこと

### 1. SimpleD&D Window 新規実装
```
src/plugins/windows/simple-dnd-window/
  GeometrySimpleDnDWindow.tsx
  CameraSimpleDnDWindow.tsx
  FxSimpleDnDWindow.tsx
  index.ts
```
- 各 ParamRow 先頭に `≡` D&D ハンドル追加
- `DragPayload` を `application/geography-param` で送信
- MacroWindow / Macro8Window の KnobCell にドロップ → AssignDialog → アサイン完了

### 2. MacroKnob ↔ D&D Window 双方向同期
- 200ms ポーリングで `engine.getParametersLive()` を使い ParamRow の value state を更新
- **「鏡」設計**: MacroKnob と D&D Window は直接交信しない。`plugin.params`（SSoT）だけを見る

### 3. MacroAssign に layerId 追加・全レイヤー同時書き込みバグ修正
- `MacroAssign` に `layerId` フィールド追加（`src/types/index.ts`）
- `AssignDialog` で `payload.layerId` を `assign` に引き継ぐ（`MacroWindow.tsx`）
- `transportManager.receiveModulation()` で `assign.layerId` で絞り込み（`transportManager.ts`）

### 4. Macro8Window 新規実装
```
src/plugins/windows/macro-8-window/
  Macro8Window.tsx
  index.ts
```
- 8ノブ・70px・4×2グリッド
- 3重リング SVG（外:青紫 / 中:緑 / 内:オレンジ）
- 隙間下側（135deg スタート）
- SVG arc largeArc 歪みバグ修正（`sweepDeg > 180` で判定）

### 5. windowMode 更新
- `GeoWindowMode`: `'none' | 'simple' | 'simple-dnd' | 'standard'`
- `MacroWindowMode`: `'none' | 'macro-window' | 'macro-8-window'`
