## 確立したアーキテクチャ（Day65）

### Window ↔ Macro の値の流れ
```
STD D&D Window（lo/hi スライダー）
  ↓ ≡ D&D
AssignDialog（assign.min/max = 正規化値）
  ↓ ASSIGN
TransportManager.receiveModulation
  → rangeMap(value, assign.min, assign.max)
  → ParameterStore
  → flushParameterStore
  → plugin.params.value（生値）
  ↓ ポーリング
STD D&D Window スライダー（lo〜hi 範囲内で表示）
```

### RangeSlider の座標系
- 上段（lo/hi）: `min〜max` 全域
- 下段（value）: `lo〜hi` 範囲内（lo=0端、hi=1端）
- MacroKnob も同じ座標系（lo=0端、hi=1端）
