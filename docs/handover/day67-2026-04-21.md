## 確定したアーキテクチャ（Day67）

```
src/
  engine/              Three.jsプラグイン（疎結合）
    geometry/
    fx/
    cameras/
    lights/
    particles/

  application/         意思決定・Registry・Transport
    command/
      geo-transitions/  GeoGraphy独自 SceneState計算型 Transition
    adapter/           入出力ドライバ（旧 drivers/）
      input/（MidiInputWrapper）
      modulator/ output/ tempo/
    schema/            型定義 SSoT（旧 types/）
      index.ts
      midi-registry.ts
      windowMode.ts
      geoAPI.d.ts

  ui/                  表示・操作
    components/
      window/
      mixers/
    panels/
    hooks/

  core/                → 将来 application/ に統合予定
  types/               → 自動生成ファイルのみ（geo-cc-map.generated.ts等）
```

### 命名の設計原則（Day67確立）
- `geo-transitions`: GeoGraphy独自（SceneState計算・Three.js不使用）→ `application/`
- GL Transitions等: Three.jsシェーダー使用 → 将来 `engine/transitions/`
- カタログ整備時に `geo-transitions` をカタログに登録予定
