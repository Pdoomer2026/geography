## 環境メモ
- **セッション開始時**：`python3 /Users/shinbigan/nfc_normalize.py` を最初に実行
- **CLAUDE.md 編集前**：`bash /Users/shinbigan/archive_claude_before_edit.sh DayN [module]`
- **引き継ぎメモ保存先**：`geography/docs/handover/dayN-YYYY-MM-DD.md`
- **dev-log 保存先**：`GeoGraphy Vault/dev-log/YYYY-MM-DD_DayN.md`
- **コミット形式**：Linus スタイル（`-m "タイトル" -m "ボディ"`）+ `git tag dayN`
- **終業時の必須手順**：引き継ぎメモ → dev-log → NFC 正規化 → git commit → tag → push
- **python3 スクリプトは直接貼り付けず**：ファイルに保存して実行
- **write_file は新規作成のみ**・edit_file は NFC 正規化済みなら安全
