# Grid Tunnel — README

## 概要

多角形リングを Z 方向に並べたトンネル。前進アニメーションでトンネルを飛び抜ける感覚を演出する。

## パラメーター

| param | 説明 | デフォルト |
|---|---|---|
| speed | 前進速度 | 0.8 |
| radius | トンネル半径 | 4.0 |
| segments | リングの多角形分割数 | 8 |
| rings | リング枚数 | 20 |
| length | トンネル全長 | 40 |
| hue | ワイヤーカラー（色相 0〜360） | 280 |

## 推奨 FX

- AfterImage: damp 0.85
- RGB Shift: amount 0.002
- Bloom: strength 0.5
